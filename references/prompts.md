## Part 1: 提示词结构

## 一、提示词核心结构

Nano Banana Pro 的最优提示词结构分为六层，按从左到右的顺序排列：

```
[画质锚定] + [主体描述] + [环境/空间] + [光影] + [构图/镜头] + [画风锚定]
```

### 各层详解

#### 第一层：画质锚定（前置权重最高）

画质锚定词决定整体生成品质，放在提示词最前面。选择取决于已确认的画风方向。

| 画质锚定词 | 适用画风方向 | 适用场景 |
|-----------|------------|---------|
| 真人实拍 | A. 真人实拍 | 真人化角色/场景 |
| 电影级摄影 | A. 真人实拍 | 真人电影画面 |
| 真人电影剧照质感 | B. 真人电影剧照 | 漫威/权游等影视级 |
| 超写实电影画质 | B/C. 电影剧照或CG | 超写实CG场景（⚠️单独使用偏CG） |
| 超逼真电影画质 | C. 3A游戏CG | 高写实CG场景 |
| 8K超清分辨率 | 通用 | 所有需要精细质感的场景 |
| 电影级镜头 | 通用 | 电影感画面 |
| 大师级CG渲染 | C/D. CG或引擎级 | 3A游戏、动画CG |
| 虚幻引擎5渲染 | D. 高精CG引擎级 | 追求接近真实的顶级CG |
| 超写实CG | D. 高精CG引擎级 | 好莱坞视效级CG |
| 电影级CG视效 | D. 高精CG引擎级 | 头号玩家/阿凡达级视效 |

**按画风方向的组合示例**：
- **真人实拍**：`真人实拍，电影级摄影，真实皮肤毛孔质感，自然光照` 
- **真人电影剧照**：`真人电影剧照质感，8K超清分辨率，电影级调色`
- **3A游戏CG**：`超写实电影画质，8K超清分辨率，电影级调色`
- **高精CG引擎级**：`虚幻引擎5渲染，8K超清分辨率，次表面散射皮肤，全局光照` 或 `超写实CG，电影级CG视效，8K超清分辨率，电影级调色`
- **偏摄影感**：`8K超清分辨率，电影级镜头`

#### 第二层：主体描述

**人物描述层次**：
```
[身份/名称] + [体态/身形] + [面部特征] + [服装/装备] + [姿态/动作] + [表情/情绪]
```

**物件/武器描述层次**：
```
[类型] + [材质] + [尺寸] + [特殊效果] + [状态]
```

#### 第三层：环境/空间

```
[空间类型] + [空间规模] + [材质/表面] + [大气效果] + [环境细节]
```

#### 第四层：光影

```
[整体明暗] + [主光源位置和类型] + [光影效果] + [色温/颜色]
```

#### 第五层：构图/镜头

在提示词中用自然语言指定构图意图和镜头参数。

#### 第六层：画风锚定（收尾兜底）

画风锚定词放在末尾，作为整体风格的兜底保障。**必须与已确认的画风方向一致。**

| 画风锚定词 | 适用画风方向 |
|-----------|------------|
| 真人实拍，电影级摄影，真实皮肤毛孔质感，自然光照 | A. 真人实拍 |
| 真人电影剧照质感，真实皮肤纹理和毛发细节 | B. 真人电影剧照 |
| 电影胶片色彩，颗粒感 | A/B. 真人方向 |
| 3A大作3D游戏风格 | C. 3A游戏CG |
| 电影级调色 | 通用 |
| 虚幻引擎5渲染，次表面散射皮肤，PBR材质 | D. 高精CG引擎级 |
| 超写实CG，电影级CG视效，电影级调色 | D. 高精CG引擎级 |
| 写意光影 | E. 特定美学 |
| 极简构图 | 通用 |

## 三、多图参考的描述规范

```
构图核心 (Composition):
前景：[角色/物体]（图1）[姿态/位置描述]。[外观细节]。
中景/背景：[角色/物体]（图2）[姿态/位置描述]。[外观细节]。
光影 (Lighting):
[光源1描述]。
[光源2描述]。
关键细节：[材质反射/倒影/特殊效果]。
镜头语言：[镜头类型]，[视角]，[景深]。
```

## 四、提示词长度控制

| 场景复杂度 | 建议长度 | 说明 |
|-----------|---------|------|
| 简单（单主体+简单背景） | 30-80字 | 如角色三视图、物件特写 |
| 中等（主体+环境+光影） | 80-150字 | 如环境概念图、单人场景 |
| 复杂（多层构图+叙事） | 150-300字 | 如对峙画面、多人场景 |

**原则**：描述精度 > 描述长度。

## Part 2: 画风锚定词库

---

## 一、画风方向选项

| 选项 | 说明 | 视觉效果 |
|------|------|---------|
| **A. 真人实拍/摄影级** | 像真实摄影照片 | 类似电影剧照、时装摄影 |
| **B. 真人电影剧照** | 介于真人和CG之间 | 类似漫威电影角色、权力的游戏 |
| **C. 3A游戏CG** | 高品质游戏CG渲染 | 类似最终幻想CG、原神过场动画 |
| **D. 高精CG引擎级** | 全CG但追求"接近真实"的顶级质感 | 类似头号玩家、虚幻引擎5 MetaHuman |
| **E. 特定美学风格** | 水墨、赛博朋克、动漫等 | 根据具体风格而定 |
| **G. 邵氏/胡金铨武侠风格** | 古典武侠、邵氏电影美学 | 类似《大醉侠》《侠女》《龙门客栈》 |

---

## 二、各方向锚定词与组合

### A. 真人实拍 / 摄影级

| 锚定词 | 适用场景 | 搭配建议 |
|--------|---------|---------|
| 真人实拍 | 直接锚定真人照片方向 | + 电影级摄影 + 自然光照 |
| 电影级摄影 | 强调摄影感而非渲染感 | + 真人实拍 |
| 真实皮肤毛孔质感 | 强制真人皮肤微观细节 | 用于角色设定图/特写 |
| RAW照片质感 | 未经后期的原始摄影感 | + 自然光照 |
| 电影胶片色彩 | 胶片感色调 | + 颗粒感 |

**组合示例**：
- `真人实拍，电影级摄影，真实皮肤毛孔质感，自然光照` — 纯真人照片级
- `真人实拍，8K超清分辨率，电影胶片色彩` — 高清真人胶片感

### B. 真人电影剧照

**组合示例**：
- `真人电影剧照质感，8K超清分辨率，真实皮肤纹理和毛发细节，电影级调色` — 影视级
- `超写实电影画质，真人电影剧照质感，电影级调色` — 超写实偏真人

### C. 3A 游戏 / CG

关键锚定词：`3A大作3D游戏风格`、`大师级CG渲染`、`超写实电影画质`、`8K超清分辨率`、`电影级调色`

### D. 高精CG引擎级

**核心定位**：全部是CG渲染，但在材质、光照、皮肤等方面无限逼近真实。与C方向的区别：C保留明显的"游戏CG美感"，D追求"以假乱真的CG"。

| 锚定词 | 适用场景 | 搭配建议 |
|--------|---------|---------|
| 虚幻引擎5渲染 | 极致光追质感 | + 全局光照 + PBR材质 |
| 次表面散射皮肤 | 皮肤半透明光感 | + 虚幻引擎5渲染 + 真实皮肤纹理 |
| PBR材质 | 物理正确的材质渲染 | + 全局光照 |
| 超写实CG | 直接锚定"追求真实感的CG" | + 电影级调色 |
| 电影级CG视效 | 好莱坞视效大片级别 | + 8K + 电影级调色 |

**组合示例**：
- `虚幻引擎5渲染，次表面散射皮肤，真实皮肤纹理，PBR材质，全局光照` — 顶级引擎级CG（人物向）
- `超写实CG，电影级CG视效，8K超清分辨率，电影级调色` — 好莱坞视效级CG（场景向）
- `光线追踪渲染，PBR材质，全局光照，电影级调色` — 引擎级CG环境

### E. 特定美学

| 锚定词 | 适用场景 |
|--------|---------|
| 中国水墨美学 | 古风/武侠/东方题材 |
| 赛博朋克美学 | 科幻/未来都市 |
| 日系动漫风格 | 二次元/番剧质感 |
| Studio Ghibli风格 | 治愈系/宫崎骏质感 |
| 低调摄影（Low-key Lighting） | 暗黑/悬疑/恐怖 |
| Golden Hour暖光 | 治愈/怀旧/情绪片 |

### G. 邵氏/胡金铨武侠风格

**核心定位**：60-70年代邵氏兄弟武侠电影美学，融合胡金铨导演的诗意武侠风格，既有戏剧化的视觉张力，又有东方古典的意境之美。

| 锚定词 | 适用场景 | 搭配建议 |
|--------|---------|---------|
| 邵氏武侠风格 | 古典武侠/古装题材 | + 胡金铨电影美学 + 邵氏电影胶片色彩 |
| 胡金铨电影美学 | 东方武侠史诗感 | + 戏剧化硬光照明 + 中式古典构图 |
| 邵氏电影胶片色彩 | 浓郁饱和的复古色彩 | + 高饱和度红色/金色/蓝色 + 胶片颗粒感 |
| 戏剧化硬光照明 | 高对比度轮廓光 | + 侧光/逆光 + 硬阴影边缘 |
| 中式古典构图 | 对称/纵深/框架构图 | + 对称构图 + 框架式前景 + 纵深引导线 |

**组合示例**：
- `邵氏武侠风格，胡金铨电影美学，邵氏电影胶片色彩，戏剧化硬光照明，中式古典构图` —— 完整邵氏武侠风格
- `电影级画质，8K超清分辨率，邵氏武侠风格，胡金铨电影美学` —— 高清邵氏武侠质感
- `邵氏电影胶片色彩，戏剧化硬光照明，高对比度，胶片颗粒感` —— 邵氏视觉风格

**风格要素详解**：

**光影特征**：
- 硬光照明，阴影边缘锐利清晰
- 高对比度，明暗反差强烈
- 侧光/逆光勾勒轮廓，正面光弱
- 戏剧化布光，类似舞台灯光效果

**色彩特征**：
- 高饱和度红色（侠客服饰、灯笼）
- 华丽金色（皇室、寺庙、装饰）
- 深邃蓝色（夜空、反派、神秘感）
- 浓郁胶片色彩，颗粒感明显
- 整体色调偏暖，复古感强烈

**构图特征**：
- 对称构图：建筑、人物、场景左右对称
- 纵深构图：利用走廊、门框、竹林形成引导线
- 框架构图：用门窗、屏风、树林作为前景框架
- 人物位置：常位于画面三分之一处或正中央

**场景元素**：
- 中式古建筑：宫殿、寺庙、客栈、城门
- 自然场景：竹林、松林、庭院、山谷
- 道具元素：红灯笼、酒旗、兵器、古筝
- 服装材质：丝绸锦缎、刺绣纹样、精致细节

**适用场景**：
- 古风/武侠/仙侠题材
- 传统工艺/东方美学
- 高端奢侈品/珠宝
- 中式餐饮/茶文化
- 文化旅游/历史品牌
| 邵氏武侠风格 | 古典武侠/古装题材 |
| 胡金铨电影美学 | 东方武侠史诗感 |
| 邵氏电影胶片色彩 | 浓郁饱和的复古色彩 |
| 戏剧化硬光照明 | 高对比度轮廓光 |
| 中式古典构图 | 对称/纵深/框架构图 |

## 三、C vs D 对比

| 维度 | C. 3A游戏CG | D. 高精CG引擎级 |
|------|------------|----------------|
| 皮肤 | 精致但光滑，有理想化美感 | 有毛孔、细纹、肤色不均，接近真实 |
| 光照 | 游戏级光照，可接受艺术化 | 全局光照+光线追踪，物理正确 |
| 材质 | 精致但允许风格化 | PBR物理正确，每个表面有真实反射/粗糙度 |
| 整体感受 | "精致的游戏角色" | "CG还是真的？" |

## 第一类：产品世界镜头

---

### 1. 产品 Hero Shot

#### 提示词模板
```
[画质锚定]，[产品描述]静置于[背景]中央。[光影设计——侧光/轮廓光/顶光]。[材质细节——金属/玻璃/磨砂]。[构图——产品占比/留白]。[画风锚定]。
```

#### 关键要点
- **背景克制**：纯色或极简影棚质感
- **光影至少双光源**：主光（侧光定型）+ 辅光（轮廓光分离背景）
- **材质必须精写**：金属拉丝、AG 磨砂玻璃、陶瓷光泽等
- **产品占比**：40%-60% 最佳

#### 示例
```

## Part 4: TVC 构图范式

TVC 的构图不仅服务于"美"，还必须服务于"传达"。

### 4.1 揭示构图

**经典揭示序列**：

```
第1步：极端微距——观众看到抽象的纹理/线条，不知道是什么
第2步：缓慢拉远——局部轮廓开始显现，观众开始猜测
第3步：继续拉远——产品全貌显现，"原来是这个"的认知快感
第4步：产品完整呈现——在完美构图中展示全貌
```

**变体手法**：遮挡揭示、光线揭示、聚焦揭示、旋转揭示。

### 4.2 End Frame 构图标准

| 元素 | 构图位置 | 尺寸要求 |
|------|---------|---------|
| **产品** | 画面中心偏左或正中 | 占画面 20%–40% |
| **品牌 Logo** | 右下角或底部居中 | 清晰可辨但不压过产品 |
| **Tagline/Slogan** | Logo 上方或产品下方 | 字号小于产品名称 |
| **背景** | 品牌色或品牌标志性视觉 | 简洁、不分散注意力 |

**End Frame 三法则**：
1. **3 秒可读**：所有信息必须在 3 秒内被观众读取完毕
2. **3 个元素以内**：产品 + Logo + Slogan 最多三个信息层级
3. **3 层视觉层级**：明确的主-次-辅视觉优先级

### 4.3 文字/Logo 安全区域

| 区域 | 定义 | 用途 |
|------|------|------|
| **画面安全区** | 边缘内缩 5% | 所有重要视觉元素 |
| **标题安全区** | 边缘内缩 10% | 所有文字信息 |
| **上方 1/5** | 顶部 20% | Tagline、标题 |
| **下方 1/5** | 底部 20% | 产品信息、法律声明 |
| **中央 60%** | 中心区域 | 核心视觉内容，避免被文字遮挡 |

---

### 4.4 剧情类构图范式

> 专门为第五类：剧情类场景设计的构图范式。通过构图来传递故事、塑造角色、增强情感。

#### 4.4.1 序幕场景构图

| 构图原则 | 说明 |
|---------|------|
| **介绍性构图** | 广角镜头，展示环境全貌 |
| **角色位置** | 主角位于画面边缘或三分法位置 |
| **环境信息** | 背景细节要丰富，建立世界观 |
| **留白暗示** | 适当留白，暗示发展空间 |

**常用构图**：
- 广角全景 → 建立环境
- 三分法构图 → 介绍主角
- 框架构图 → 暗示主角被环境包围

---

#### 4.4.2 冲突场景构图

| 构图原则 | 说明 |
|---------|------|
| **压迫性构图** | 低角度、倾斜构图、近距离特写 |
| **角色位置** | 主角被挤压在画面边缘 |
| **障碍物前景** | 用障碍物作为前景，增加困境感 |
| **不平衡构图** | 打破对称，营造紧张感 |

**常用构图**：
- 低角度仰拍 → 增强压迫感
- 倾斜构图（Dutch Angle）→ 增加不安感
- 框架构图（主角被困）→ 暗示困境
- 近距离特写 → 放大情绪张力

---

#### 4.4.3 发现场景构图

| 构图原则 | 说明 |
|---------|------|
| **揭示性构图** | 从局部到整体，从暗到明 |
| **产品位置** | 产品逐渐成为画面中心 |
| **引导线** | 用视线、光线、线条引导向产品 |
| **亮暗对比** | 产品区域亮，周围暗 |

**常用构图**：
- 揭示序列（微距 → 拉远 → 全貌）→ 经典揭示
- 视线引导构图 → 主角看什么，观众看什么
- 光线揭示 → 光线逐渐照亮产品
- 中心构图 → 产品最终位于画面中央

| **角色位置** | 主角与对手/困境分置画面两侧 |
| **张力线** | 用视线、动作线建立张力 |
| **动态构图** | 低角度、倾斜、运动模糊 |

**常用构图**：
- 对称对峙构图 → 主角与对手各占一侧
- 对角线张力构图 → 增强对决感
- 低角度仰拍 → 史诗感、力量感
- 过肩视角（OTS）→ 主观对决感

### 4.5 电影类构图范式

> 借鉴经典电影语言的构图范式。适用于所有场景类型，特别是追求电影感的 TVC。

#### 4.5.1 经典电影构图法则

**法则 1：三分法（Rule of Thirds）**
- 将画面分为 3×3 网格
- 重要元素位于交叉点上
- 比中心构图更有活力

**法则 2：对称构图（Symmetry）**
- 左右或上下完全对称
- 适用于仪式感、史诗感、稳定感
- 邵氏武侠风格常用

**法则 3：框架构图（Frame within a Frame）**
- 用门窗、屏风、树林作为前景框架
- 增加层次感、深度感
- 引导观众视线

**法则 4：引导线构图（Leading Lines）**
- 用建筑、道路、光线等线条引导视线
- 纵深构图常用
- 增强空间感

**法则 5：留白构图（Negative Space）**
- 大面积空白区域
- 突出主体，增加高级感
- End Frame 必须预留文字空间

---

#### 4.5.2 镜头类型与构图

**镜头类型** | **构图特点** | **适用场景**
---|---|---
**广角镜头（Wide）** | 夸张透视，容纳更多环境 | 序幕场景、环境空镜、宏大场景
**标准镜头（Normal）** | 自然透视，接近人眼 | 中景、人物对话、日常场景
**长焦镜头（Telephoto）** | 压缩空间，突出主体 | 特写、产品展示、对决场景
**微距镜头（Macro）** | 极端特写，微观细节 | 产品微距、材质展示、揭示序列

---

#### 4.5.3 拍摄角度与构图

**拍摄角度** | **构图效果** | **情感暗示**
---|---|---
**平视角度（Eye Level）** | 自然、平等、真实 | 日常场景、对话、真实感
**低角度仰拍（Low Angle）** | 主体高大、有力量、史诗感 | 英雄登场、对决场景、宫殿仪式
**高角度俯拍（High Angle）** | 主体渺小、脆弱、被观察 | 冲突场景、困境展示、追踪场景
**鸟瞰角度（Bird's Eye）** | 上帝视角、全局观察 | 环境总览、追踪场景、山谷追踪
**倾斜角度（Dutch Angle）** | 不安、紧张、失控 | 冲突场景、对决场景、紧张时刻

---

#### 4.5.4 景别与构图

**景别** | **画面内容** | **构图要点**
---|---|---
**特写（Close-Up, CU）** | 肩部以上或局部 | 背景干净，突出细节，产品微距
**中特写（Medium Close-Up, MCU）** | 胸部以上 | 表情+环境平衡，情绪传递
**中景（Medium Shot, MS）** | 腰部以上 | 动作+表情，人物关系
**中全景（Medium Long Shot, MLS）** | 膝盖以上 | 人物+环境，动作展示
**全景（Long Shot, LS）** | 全身 | 人物+完整环境，关系建立
**大全景（Wide Shot, WS）** | 环境全貌 | 建立世界观，宏大场景

---

#### 4.5.5 邵氏武侠电影构图范式 ⭐

**核心特点**：
1. **绝对对称构图** - 建筑、人物、场景严格对称
2. **纵深引导线** - 走廊、门框、竹林形成引导线
3. **框架式前景** - 门窗、屏风、树林作为框架
4. **戏剧化硬光** - 低角度侧光，阴影锐利
5. **人物位置** - 三分法或正中央

**经典邵氏构图**：
- **竹林对决**：纵深引导线 + 框架构图 + 低角度侧光
- **客栈密谋**：框架式前景 + 对称构图 + 灯笼暖光
- **宫殿仪式**：绝对对称 + 顶光+侧光 + 宏大纵深感
- **山谷追踪**：低角度俯拍 + 纵深引导线 + 雾气层次

真人实拍，电影级摄影，8K超清分辨率。第二天清晨的城市街头，阳光明媚。那位年轻人身着笔挺的西装，领带整齐，精神焕发，完全不像昨天的狼狈模样。他站在街头，抬起左手腕，全新的智能手表在阳光下闪闪发光，屏幕显示着今日日程和运动目标。背景中，曾经拥挤的地铁依然存在，但年轻人已经不再需要挤地铁——他选择了跑步上班，智能手表精准记录着他的运动数据。早晨的金色阳光从左侧洒下，在智能手表的金属表壳上形成耀眼的高光。低角度仰拍，年轻人位于画面左侧，智能手表在晨光中熠熠生辉。暖色调，高对比度，电影级调色。
```

### 21. 结局场景

#### 提示词模板
```
[画质锚定]，[结局环境描述——时间/地点/氛围]。[主角描述——成长/蜕变/满足]。[产品在结局中的角色——生活的一部分]。[情感升华——微笑/满足/希望]。[光影——温暖/明亮/充满希望]。[构图——稳定/圆满]。[画风锚定]。
```

#### 关键要点
- **情感落地**：问题解决，情绪满足
- **产品融入**：产品不再是"解决方案"，而是生活的一部分
- **成长弧光**：展示主角从序幕到结局的成长和蜕变
- **预留空间**：End Frame 要为 Logo 和 Slogan 预留空间

#### 示例
```
真人实拍，电影级摄影，8K超清分辨率。傍晚的城市公园，夕阳西下，金色的阳光洒满草地。那位年轻人坐在长椅上，身着休闲装，完全放松，嘴角带着满足的微笑。他的左手腕上，智能手表依然佩戴着，屏幕显示着今日完成的运动圆环，全部闭合。旁边放着一杯咖啡，热气缓缓升起。背景是城市天际线的剪影，夕阳的金色光芒在天际线上渐变。中景平视，年轻人位于画面左侧三分之一处，右侧三分之二预留出干净空间。暖金色调，柔和的夕阳光影，电影级调色。
```

## Appendix: TVC 视觉设计清单

- [ ] 品牌色是否有明确的植入策略？
- [ ] 品牌色是否与全片色彩弧线自然融合？
- [ ] 产品材质之美是否通过正确的光线策略被释放？
- [ ] 产品镜头是否有"雕塑化"处理？
- [ ] 产品与负空间的比例是否服务于叙事意图？
- [ ] 构图是否为文字/Logo 预留了安全区域？
- [ ] End Frame 是否满足"3 秒可读、3 个元素、3 层层级"？
- [ ] 是否确定了品牌所属的视觉世界并贯穿全片？
- [ ] 视觉世界的选择是否匹配品牌调性和目标受众？
